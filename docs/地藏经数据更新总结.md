## 使用方法

在应用中导入数据：

```typescript
import { dizangSutraTitle, dizangChapters, dizangSections } from './data/dizang';

// 获取经文标题
console.log(dizangSutraTitle); // "地藏菩萨本愿经"

// 获取章节列表
console.log(dizangChapters); // 十三品的章节信息数组

// 获取所有段落
console.log(dizangSections); // 所有经文段落数组
```
