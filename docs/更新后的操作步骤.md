## 需要执行的操作

### 1. 重启开发服务器

由于数据文件已更新，需要重启开发服务器来加载新数据：

```bash
# 在 prajna-buddy 目录下执行
cd prajna-buddy
npm run dev
```

或者在项目根目录执行：
```bash
npm run dev
```

### 2. 清除浏览器缓存（可选）

如果重启后仍然看不到更新，可以尝试：
1. 按 `Ctrl + Shift + R` (Windows) 或 `Cmd + Shift + R` (Mac) 强制刷新
2. 或者打开浏览器开发者工具，右键刷新按钮，选择"清空缓存并硬性重新加载"
