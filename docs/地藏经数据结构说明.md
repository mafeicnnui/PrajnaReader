# 地藏经数据说明

## 文件结构

- `dizang.ts` - 主导出文件，供应用使用
- `dizang-full.ts` - 完整的地藏经数据，包含十三品的经文内容

## 数据格式

### ChapterInfo (章节信息)
```typescript
{
  chapterId: number;        // 章节编号 (1-13)
  chapterTitle: string;     // 章节标题
  description: string;      // 章节简介
}
```

### SutraSection (经文段落)
```typescript
{
  id: string;              // 唯一标识，如 'ch1-1'
  title?: string;          // 段落标题（可选）
  text: string;            // 经文原文
  pinyin: string;          // 拼音注音
  meaning: string;         // 白话文解释
}
```

## 如何扩展内容

1. 打开 `dizang-full.ts` 文件
2. 在对应的 `chapterXSections` 数组中添加新段落
3. 按照 `SutraSection` 格式填写内容
4. 拼音和白话文可以逐步补充

## 数据来源

经文原文来自公开的佛经资源，遵循佛教经典的传统版本。

## 注意事项

- 拼音注音需要专业人士校对
- 白话文翻译应准确传达经文含义
- 保持经文的完整性和准确性
- 尊重佛教经典的神圣性
