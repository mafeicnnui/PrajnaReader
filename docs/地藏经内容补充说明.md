## 如何获取完整经文

由于版权限制，建议从以下合法渠道获取地藏经完整内容：

### 推荐资源

1. **佛光山电子大藏经**
   - 网址：https://etext.fgs.org.tw
   - 提供免费的佛经电子版本
   - 包含经文、注音和白话文

2. **CBETA 中华电子佛典协会**
   - 网址：http://www.cbeta.org
   - 提供大正藏第十三册内容
   - 地藏经由唐代实叉难陀译

3. **Chinese Text Project (ctext.org)**
   - 网址：https://ctext.org
   - 大型中文古籍数字图书馆
   - 可能包含佛经内容

## 数据格式说明

每个段落需要包含以下字段：

```typescript
{
  id: string;           // 唯一标识，如 'chapter-2-1'
  title?: string;       // 段落标题（可选）
  text: string;         // 经文原文
  pinyin: string;       // 拼音注音
  meaning: string;      // 白话文解释
}
```
