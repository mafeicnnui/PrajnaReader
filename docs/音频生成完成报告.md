## 技术细节

### 生成脚本

1. **export_sections.py**
   - 从 `dizang-full.ts` 提取 section 数据
   - 生成 `sections.json` 文件
   - 提取 id, title, text 字段

2. **generate_sections.py**
   - 读取 `sections.json`
   - 调用 Edge TTS API 生成音频
   - 支持男声和女声
   - 自动跳过已存在的文件
   - 支持增量生成

### 命令示例

```bash
# 导出数据
python export_sections.py

# 生成所有音频
python generate_sections.py

# 只生成男声
python generate_sections.py --voice male

# 只生成特定 sections
python generate_sections.py --sections ch1-1 ch1-2 ch1-3

# 使用代理
python generate_sections.py --proxy http://127.0.0.1:7890
```
